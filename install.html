Installation of Election Poll

 

Welcome to Election Poll. Before to start, we need some information of your data base. You will need to know the next before to cotinue.

  • Name of Data Base
  • User of Data Base
  • Password of Data Base
  • Server of Data Base

We will use that information to create config.php file. If for any reason doesn't work the automatic creation of this file, don't worry! This file only contains the information of your data base in a configuration file.

Probably this elements has provided your hosting manager. If you haven't this information, you will have to put in contact with them to continue. Now, you're ready!
